# NU·OIR LABS | OSSD Documentation Hub

This folder contains all supporting materials for the Omnidirectional Superconducting Spherical Drivetrain (OSSD).

### Contents
- **Schematics_v1.pdf** – Engineering diagrams and component layout  
- **CryoCaster_Prototype_Notes.md** – Notes on cryogenic containment and flux-pinning results  
- **Material_Data.xlsx** – Magnetic field strength vs temperature chart  
- **Photos/** – Prototype and lab imagery  

### Versioning
Each document will be version-controlled with changelogs for iterative testing and grant submission requirements.
